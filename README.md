# ASSA SPOR Bale Barı Kataloğu

Bu proje, ASSA SPOR bale barı modelleri için hazırlanmış tek sayfalık dijital katalog ve WhatsApp teklif alma sayfasıdır.

## İçerik

- Ayarlanabilir bale barları
- Duvara monte bale barları
- Taşınabilir modeller
- Sabit yükseklik modelleri
- WhatsApp üzerinden seçili ürünler için bilgi alma sistemi

## Yayınlama

Bu proje GitHub Pages üzerinden yayınlanabilir.
GitHub repo ayarlarından:

Settings → Pages → Branch: main → Folder: /root

seçilerek canlıya alınabilir.
